Point Entity

AI Goal Standoff

Keyvalues

  • Actor(s) to affect (actor) <TargetNameOrClass>

  • Search Type (SearchType) <Choices>
    How to search for the entities using the targetname.

    • Entity Name
    • Classname

  • Start Active (StartActive) <Boolean>

  • Reaction to tactical change (HintGroupChangeReaction) <Choices>
    What to do if leader moves, threat is neutralized, hint group changes, etc

    • Move when ready (default AI)
    • Move when seek cover
    • Move immediately

  • Aggressiveness (Aggressiveness) <Choices>

    • Very low
    • Low
    • Medium
    • High
    • Very High

  • Player battleline (PlayerBattleline) <Boolean>
    Player defines a battle line, applies to allies only

  • Stay at cover location (StayAtCover) <Boolean>
    When have suitable cover, don't change it (disables advancing to battle line)

  • Abandon if enemies hide (AbandonIfEnemyHides) <Boolean>
    If no enemy detected recently, stop the standoff

Inputs

  • Activate <Void>
    Begin contesting position

  • Deactivate <Void>
    Cease contesting position

  • SetAggressiveness <Integer>
    Set aggressiveness
