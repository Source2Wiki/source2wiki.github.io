Point Entity

Combination of an env_cubemap_box and an env_light_probe_volume

Keyvalues

  • Priority (indoor_outdoor_level) <Choices>
    If multiple volumes contain an object, the highest priority volume takes precedence.

    • Lowest
    • Low
    • Normal
    • High
    • Highest

  • Light Probe Texture (lightprobetexture) <Resource>
    Name of the light probe texture

  • Box Mins (box_mins) <Vector>

  • Box Maxs (box_maxs) <Vector>

  • Ignore Unreachable Space (flood_fill) <Boolean>

  • Calculate Diffuse Lighting Using Cubemap (light_probe_volume_from_cubemap) <Boolean>

  • Storage (storage) <Choices>

    • Default
    • AmbientCube RGBM8888 Uncompressed
    • AmbientCube RGBMDXT5 Compressed

  • Cubemap Texture (cubemaptexture) <Resource>
    Name of the cubemap texture

  • Bake Near Z (bakenearz) <Float>
    Near clip plane used for the camera when baking the cube map

  • Bake Far Z (bakefarz) <Float>
    Far clip plane used for the camera when baking the cube map

  • Light Group (lightgroup) <String>
    Semicolon-delimited list of light groups to affect

  • Moveable (moveable) <Boolean>
    Can be moved, parented, enabled, or disabled at runtime.

  • Voxel Size (voxel_size) <Choices>

    • Very High Resolution
    • High Resolution
    • Medium Resolution
    • Low Resolution
    • Very High Resolution
    • High Resolution
    • Medium Resolution
    • Low Resolution

Inputs

  • SetColor <Vector>
    Set color

  • SetBrightness <Float>
    Set brightness
