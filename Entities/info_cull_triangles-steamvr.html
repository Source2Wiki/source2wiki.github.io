Point Entity

Keyvalues

  • Box Size (box_size) <Vector>

  • Limit to Prefab (limit_to_world) <Boolean>
    Limit the box to culling triangles objects in the prefab to which it belongs.

  • Target Objects (targets) <NodeIdList>
    Limit the box to culling triangles from the specified target objects.

  • Apply To (geometry_type) <Choices>
    Spcifies the type of geometry which can be culled by the box

    • Everything
    • Only Static Props
    • Only World Geometry
