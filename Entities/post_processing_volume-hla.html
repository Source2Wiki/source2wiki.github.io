Mesh Entity

Set postprocessing params while a player is inside the volume.

Keyvalues

  • Postprocessing Resource Name (postprocessing) <Resource>

  • Fade Time (fadetime) <Float>
    Time to fade to these postprocessing settings in seconds

  • Exposure Control (enableexposure) <Boolean>
    Use exposure settings for this volume?

  • Min Exposure Brightness (minexposure) <Float>
    Minimum autoexposure scale

  • Max Exposure Brightness (maxexposure) <Float>
    Maximum autoexposure scale

  • Exposure Compensation (exposurecompensation) <Float>
    Number of stops to adjust auto-exposure by

  • Exposure Adjustment Speed Up (exposurespeedup) <Float>

  • Exposure Adjustment Speed Down (exposurespeeddown) <Float>

  • Master Volume (master) <Boolean>
    Is this volume active by default even when not touched by the player? (Master)
