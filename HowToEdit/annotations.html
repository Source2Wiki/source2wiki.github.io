Annotations

An annotation, useful for adding extra information, use it by wrapping text with a set of 3 colons, followed by a label denoting its type:

:::note
This is a `note` Admonition.
:::
note

This is a note Admonition.

tip

This is a tip Admonition.

info

This is an info Admonition.

warning

This is a warning Admonition.

danger

This is a danger Admonition.

[LEGACY]
This Page contains Legacy information that is only here for parity.
Non FGD
This page is for an entity which exists in engine code, but not in the FGD, you will need to change the classname of the entity and add the keyvalues manually in Hammer!
